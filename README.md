# Food in Vale Dataset

Dataset para o app **Food in Vale**.

## Como usar

Para usar é simples:

-   Acesse o arquivo `.json` que você fara a requisição
-   Acesse o `raw`
-   Copie e URL
-   Use a URL para fazer uma requisição HTTP
